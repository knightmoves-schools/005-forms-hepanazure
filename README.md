# 005 Forms

[![005 Forms](https://img.youtube.com/vi/EuIFbWgbTlg/0.jpg)](https://www.youtube.com/watch?v=EuIFbWgbTlg)

The `index.html` file
- should contain a form 
- should contain 2 radio buttons with labels and one of them selected by default inside the form
- should contain 3 checkboxes with labels and one of them checked by default inside the form
- should contain 1 input with a placeholder and that is marked as required inside the form
- should contain 1 submit button inside the form
